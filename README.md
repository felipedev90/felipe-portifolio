Este projeto é o meu **portfólio pessoal como Desenvolvedor Front-End**, criado para apresentar meus principais projetos, habilidades e evolução ao longo da minha jornada no desenvolvimento web.

O objetivo é demonstrar boas práticas de desenvolvimento, organização de código e construção de interfaces modernas e responsivas.

## Funcionalidades

- Layout responsivo (Mobile First)
- Estrutura HTML semântica
- Estilização com CSS Modules
- Formulário de contato com validação (React Hook Form + Zod)
- Integração com Web3Forms para envio de mensagens
- Testes automatizados com Vitest e Testing Library
- Pipeline CI/CD com GitHub Actions
- Organização de código voltada para escalabilidade
- Interface otimizada para desktop e dispositivos móveis

## Estrutura do projeto

```
felipe-portifolio/
├── .github/
│   └── workflows/      # Configuração do GitHub Actions (CI/CD)
├── public/             # Imagens utilizadas no portfólio
│   └── projects/
└── src/
    ├── components/     # Componentes React
    │   ├── About/
    │   ├── Contact/
    │   ├── Footer/
    │   ├── Hero/
    │   ├── NavBar/
    │   ├── Projects/
    │   └── Skills/
    ├── data/           # Dados dos componentes (About, Projects, Skills)
    ├── schemas/        # Validação do formulário com Zod
    ├── styles/         # Estilos globais
    └── test/           # Setup do Testing Library
```
